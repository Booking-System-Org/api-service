# API Service

Микросервис для управления бронированиями в системе ресторанов. Предоставляет REST API для создания и получения информации о бронированиях

## Быстрый старт

### 1. Первый запуск

```bash
make provision
```

### 2. Остановка

```bash
make stop
```

### 3. Очистка

```bash
make clean
```
